> 參考：[[頂級網站技術長高度：前端工程進階大師指南]] 的第四章。

- 函數參數為原始型別時，函數本身會複製一份參考值，任何操作都不會影響原參數的實際值。
- 函數參數為參考類型時，當在函數中修改參數的某個屬性的值時，將對原來的參數進行修改。
- 函數參數為參考類型時，如果直接修改參數的參考位置，則相當於在函數中新建立一個參考，任何操作都不會影響原參數的實際值。
	```js
	function boo(val) {
	    val = {age: 200}
	}
	const coo = {tel: '00'}

	boo(coo)
	console.log(coo) // {tel: '00'}
	```
